# 五子棋 - AI 对战

基于浏览器的五子棋游戏，支持人机对战，无需后端。

## 玩法说明

1. **选择难度**：低/中/高，影响 AI 搜索深度和策略。
2. **选择执棋**：执黑先手或执白后手。
3. **开始游戏**：点击「开始游戏」按钮。
4. **落子**：点击棋盘空格落子，桌面端鼠标悬停有预览。
5. **提示**：中/高难度可使用「💡提示」按钮获得 AI 推荐落子（高难度 2 次，中难度 1 次）。
6. **悔棋**：通过连败彩蛋获得悔棋次数（1 或 2 次），撤回玩家和 AI 各一步。
7. **连败彩蛋**：连续输给 AI 3 局后触发彩蛋面板，可选择：
   - 再给点提示：额外 +1 次提示机会
   - AI 放放水：下一局 AI 变低难度
   - 悔一步/悔两步：下一局获得悔棋机会
   - 幸运星指引：下一局开盘 3 颗金星闪烁
   - 棋盘恶作剧：下一局棋盘变粉红色
8. **胜利特效**：五连获胜后棋子变色抖动，烟花绽放。
9. **平局猫**：平局时显示猫咪彩蛋。

## 技术栈

- HTML / CSS / JavaScript (TypeScript)
- React 18 + Vite
- Zustand 状态管理
- Tailwind CSS 样式
- AI 引擎：Minimax + Alpha-Beta 剪枝，纯前端计算
- 无需后端，纯静态文件部署

## 本地开发

```bash
npm run dev
```
